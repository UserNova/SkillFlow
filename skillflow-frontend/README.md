
  # Web Dashboard UI Design

  This is a code bundle for Web Dashboard UI Design. The original project is available at https://www.figma.com/design/uDvvvXvcdhnLfsFE0M6RZo/Web-Dashboard-UI-Design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  

## Configuration API

Le frontend utilise `VITE_API_BASE_URL` (fichier `.env`) pour pointer vers le Gateway.

Exemple :

```bash
VITE_API_BASE_URL=http://localhost:8888
```
